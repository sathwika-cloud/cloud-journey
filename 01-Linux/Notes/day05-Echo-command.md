# Linux Echo Command

## What is `echo`?

The `echo` command displays text in the terminal or writes text to a file.

## Examples

Print text:

```bash
echo "Hello"
```

Overwrite a file:

```bash
echo "Hello" > notes.txt
```

Append to a file:

```bash
echo "Cloud Engineer" >> notes.txt
```

## Difference

- `>` : Overwrites the existing file content.
- `>>` : Appends text to the end of the file.

## Why Cloud Engineers use `echo`

- Display status messages.
- Write configuration values.
- Create test files.
- Debug shell scripts.
